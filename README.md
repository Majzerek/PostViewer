# a mini-app for browsing posts from JSONPlaceholder with search, author filter, post details and a "Favorites" list.

# Using React, Vite, Mui, Playwright# PostViewer

### husky before start for dev mode;

- npx husky init
- npx lint-staged
- npm run prepare

### after start

- npm run build
- npm run start
- npm run test
  or
- npm run test:ui
